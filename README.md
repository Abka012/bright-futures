# Welcome to your Bright Futures

## Project info

Bright Futures currently relies on multiple spreadsheets and manual record-keeping to manage information about schools, volunteers, tour schedules, and partner organizations. 
This approach is time-consuming, prone to errors, and makes it difficult to generate reports or track engagement effectively.

The proposed solution is a centralized web-based management system that securely stores all operational data in one place.
The system will streamline scheduling, reduce data duplication, improve communication among stakeholders, and provide reporting tools to support better decision-making and overall efficiency.

## How to run project locally

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
